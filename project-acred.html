Project information

  • Release date: Work In Progress
  • Project URL: Assassin's Creed Codename Red - Reveal Trailer | Ubisoft Forward 2022

Key Contributions

  • Refined and developed new gameplay features for both Fight Systems and Animation Calibration Systems .
  • Integrated existing codebases from previous projects, optimizing them for the Multiple Games Pipeline.
  • Developed rope physics simulations for chain weapon visual and gameplay mechanics.


    • Fight systems - A highly data driven system handles all the features related to combat, such as hurtbox, hitbox, combat states, AI, Animations etc. For more informaion

    Animation Calibration Systems - This system dynamically adjusts the timing and orientation of animations in real-time based on terrain or gameplay requirements. Primarily utilized in combat and parkour scenarios.

    Multiple games pipeline - Sharing the majority of the code across multiple games in development. Utilizing a highly data-driven approach that empowers designers and artists to craft unique games solely through data.
